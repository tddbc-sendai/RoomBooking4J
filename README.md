RoomBooking4J
=============

## 課題：会議室予約システム

- 課題１ １つの会議室の予約
    - 会議室を予約できる
    - 会議室が予約済か確認できる
- 課題２ 予約済の確認とキャンセル
    - 予約済の会議室は予約できない
    - 予約済の会議室をキャンセルできる
    - キャンセルした会議室は再予約できる
- 課題３ 日付の指定
    - 日付を１日指定して会議室を予約できる
    - 日付を１日指定して会議室が予約済か確認できる
    - 日付を１日指定して予約済の会議室をキャンセルできる
- 課題４ 連続した日付の指定
    - 連続した日付を指定して会議室を予約できる
